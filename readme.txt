放置ゲーム(仮) 

- 基本ルール
1. 空から色々な「モノ」が降ってきます。
2. 同じ種類の「モノ」同士をくっ付けると一つ上のランクの別の「モノ」に変化します。
3. 「モノ」は一定時間間隔でお金を稼ぎます。ランクが高い「モノ」ほどお金を稼ぐ能力は高くなります。
4. 稼いだお金を使ってレベルを上げられます。レベルを上げると降ってくる「モノ」のランクが一段階上がります。
5. 「モノ」の合体、お金稼ぎ、レベル上げを繰り返して全ての「モノ」をコンプリートしましょう。

- ガチャカプセル
ランクの低い「モノ」はリサイクルに出すことができます。
5個リサイクルするたびに、ランダムな「モノ」が入っているガチャカプセルを1個入手できます。
ガチャをうまく利用して高ランクの「モノ」を手に入れましょう。
ちなみに、低確率でいいことが起こります。

- コンボシステム
「モノ」同士を合体させた時、一定量のお金が手に入ります。
このお金はコンボを決めることによってもらえる量が増加します。
コンボは、連続で同じ「モノ」に対して合体し続けた場合に発生します。
たくさんコンボできるように「モノ」の配置を考えるのも大事になります。

- 操作方法
マウスだけを利用します。
ドラッグ...「モノ」の移動
右クリック...リサイクル
マウスホイール...スクロール
